# 🧮 Calculator App

A simple and modern calculator built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations with a clean and responsive user interface.

---

## 🚀 Features

- ➕ Addition, Subtraction, Multiplication, Division
- 🎨 Modern UI with gradient design
- 🖱️ Button click interactions with animations
- 📱 Responsive layout (works on different screen sizes)
- ⚡ Instant calculation using JavaScript logic

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)

---

## 📌 What I Learned

- Event handling in JavaScript
- DOM manipulation
- Building UI layouts using CSS Grid/Flexbox
- Handling user input dynamically
- Error handling in basic expressions
