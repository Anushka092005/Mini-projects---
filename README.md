# 🚀 Mini Projects Collection

This repository contains a collection of beginner-friendly projects built using Python, HTML, CSS, and JavaScript. These projects were created to strengthen my fundamentals in programming, problem-solving, and frontend development.

---

## 📌 Projects Included

### 📝 1. To-Do App
A modern task management web application built using HTML, CSS, and JavaScript.

**Features:**
- Add and delete tasks
- Mark tasks as completed
- Task categories (Study, Work, Personal)
- Filter tasks (All / Active / Completed)
- Local storage support (data persists after refresh)
- Smooth UI animations

---

### 🧮 2. Calculator
A responsive and modern calculator built using HTML, CSS, and JavaScript.

**Features:**
- Basic arithmetic operations (+, -, *, /)
- Clean and modern UI design
- Button hover and click animations
- Responsive layout

---

### 🎯 3. Number Guessing Game (Python)
A simple console-based game where the user tries to guess a randomly generated number.

**Features:**
- Random number generation
- User input handling
- Hints (too high / too low)
- Beginner-friendly logic practice
